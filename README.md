# 📊 E-Commerce Business Intelligence Dashboard

## 📌 Project Overview
This Power BI project analyzes e-commerce sales data to provide actionable business insights related to sales performance, product profitability, and customer behavior. The dashboard is designed to help stakeholders make data-driven decisions through interactive visualizations and KPI tracking.

---

## 🎯 Objectives
- Monitor overall business performance using key KPIs.
- Identify top-performing products and profitable categories.
- Analyze customer segments and regional trends.
- Enable interactive exploration of data through slicers and navigation buttons.

---

### 1. Executive Overview

- YoY Growth

- Total Quantity Sold

- Sales by Category
- Profit by Region

### 2. Product Analysis
- Average Selling Price
- Average Profit per Order

### 3. Customer Analysis
- Total Customers
- Average Order Value
- Average Orders per Customer
- Sales by Segment
- Profit by Region
- Sales by State
- Top Customers by Sales
- Sales vs Profit by Region
